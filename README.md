# 🐱 GatoMia - Frontend em Angular

>Versão 0.0.1 - Primeiros Componentes e Layout Base

## 📋 Sobre o Projeto

**GatoMia** é um projeto acadêmico desenvolvido como parte do currículo da faculdade. Originalmente implementado em **React**, esta versão em **Angular** tem como objetivo aprender e explorar os conceitos fundamentais do framework Angular 21.2.

## 🚀 Começando

### Pré-requisitos

- Node.js (v18 ou superior)
- npm (v11.6.2 ou superior)

### Instalação

1. **Clonar o repositório** (se aplicável)
```bash
git clone https://github.com/VictorPortolani/frontend-GatoMia-Angular
cd frontend-gatomia
```

2. **Instalar dependências**
```bash
npm install
```

3. **Iniciar o servidor de desenvolvimento**
```bash
npm start
```

A aplicação estará disponível em `http://localhost:4200/`
